# {{projectName}}

An Expo app scaffolded with [FusionUI](https://rukkiecodes.github.io/fusionui/).

```bash
npm install
npm run start
```

Components come from `@fusionui/native`; the design tokens from `@fusionui/tokens`.
The liquid-glass surface uses `@shopify/react-native-skia` (and the real
`UIGlassEffect` on iOS 26 if `expo-glass-effect` is installed).
