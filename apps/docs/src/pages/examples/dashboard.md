# Dashboard

A realistic composition — stat cards, a live area chart, and an activity feed —
built entirely from FusionUI components and tokens. This is the "documented by
example" reference: copy the pattern, swap your data.

<DashboardExample />

It uses `FCard`, `FLineChart` (monotone area, the honest curve), `FChip`,
`FBtn`, `FAvatar` and `FIcon` — all reading the same design tokens, so it
re-themes (light/dark or a brand) without touching this markup. Toggle the theme
in the top bar to see it adapt.
