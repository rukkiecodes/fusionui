# FusionUI

A Vue 3 component framework with the engineering stability of
[Vuetify](https://vuetifyjs.com) and the visual polish of
[Vuesax](https://vuesax.com), powered by the built-in FusionUI icon set.

<Example file="button/variants" />

## Why FusionUI?

- **Stable architecture** — Vue 3 + TypeScript, composable-driven, CSS-variable theming.
- **Vuesax looks** — soft shadows, gentle radii, smooth transitions, ripples, and lift-on-hover.
- **FusionUI icons** — 737 tree-shakeable icons ship by default, and growing.
- **Batteries included** — buttons, cards, inputs, selection controls, overlays, and more.
