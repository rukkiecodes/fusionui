# Button

`VdBtn` is the flagship component, with the full set of Vuesax v4 button looks —
softened with Apple-style spacing and the Vue DL color system.

## Variants

Ten variants, from solid `elevated` to the elevated `floating`, the tinted
`tonal`, the diagonal `gradient`, the embossed `relief`, and the neutral
`shadow`.

<Example file="button/variants" />

## Sizes, icons, loading & hover animation

Five sizes, icon-only and `circle` buttons, a built-in loading state, and an
`animate` slot that swaps in on hover.
