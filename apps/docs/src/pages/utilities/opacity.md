# Opacity

Fade an element with `opacity-0` through `opacity-100`, in steps of 10.

<div class="u-demo d-flex ga-3 flex-wrap align-center">
  <span class="u-block opacity-10 d-flex align-center justify-center" style="width:60px;height:60px;color:#fff;font-size:12px">10</span>
  <span class="u-block opacity-30 d-flex align-center justify-center" style="width:60px;height:60px;color:#fff;font-size:12px">30</span>
  <span class="u-block opacity-50 d-flex align-center justify-center" style="width:60px;height:60px;color:#fff;font-size:12px">50</span>
  <span class="u-block opacity-70 d-flex align-center justify-center" style="width:60px;height:60px;color:#fff;font-size:12px">70</span>
  <span class="u-block opacity-100 d-flex align-center justify-center" style="width:60px;height:60px;color:#fff;font-size:12px">100</span>
</div>

```html
<div class="opacity-50">half-faded</div>
<button class="opacity-40">looks disabled</button>
```

## Reference

`opacity-0` · `opacity-10` · `opacity-20` · … · `opacity-90` · `opacity-100`
— maps to `opacity: 0` … `1`.
